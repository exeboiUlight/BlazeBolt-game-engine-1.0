BlazeBolt Lua API 1.0

Полная документация по Lua API игрового движка BlazeBolt 1.0

◆Система координат

Движок использует стандартные координаты OpenGL NDC (Normalized Device Coordinates)

Y ^ (-1,1) -----+----- (1,1) | | | | ------+------> X | (0,0)| | | (-1,-1)-----+----- (1,-1)
  • (0, 0) — центр экрана
  • (1, 1) — правый верхний угол
  • (-1, -1) — левый нижний угол
  • (1, -1) — правый нижний угол
  • (-1, 1) — левый верхний угол
Совет: Размеры задаются в NDC: 0.1 = 10% от высоты экрана.

Конвертация пикселей в NDC

lua 
function pixelsToNDC(px, py)
    local ndcX = (px / GetScreenWidth()) * 2 - 1
    local ndcY = 1 - (py / GetScreenHeight()) * 2
    return ndcX, ndcY
end

◆Жизненный цикл

Callbacks вызываются движком автоматически

Основные callbacks

lua 
function Start()       -- вызывается один раз при запуске
function Update(dt)    -- вызывается каждый кадр (dt в секундах)
function Draw()        -- вызывается для отрисовки каждый кадр
function End()         -- вызывается при завершении работы

Callbacks сцен

lua 
function OnMenuLoad()    -- при загрузке сцены "Menu"
function OnMenuUnload()  -- при выгрузке сцены "Menu"

Имя функции формируется как On<SceneName>Load / On<SceneName>Unload.

◆Спрайты

2D изображения с текстурами, позицией, размером и поворотом

Создание

lua 
entity = BlazeBolt.CreateSprite(texturePath, x, y)
ПараметрТипОписание
texturePathstringПуть к файлу текстуры (PNG)
xnumberПозиция X в NDC
ynumberПозиция Y в NDC

Возвращает: entity (integer) — идентификатор объекта

Позиция

lua
BlazeBolt.SpriteSetPosition(entity, x, y)
x, y = BlazeBolt.SpriteGetPosition(entity)

Размер

lua
BlazeBolt.SpriteSetSize(entity, width, height)
width, height = BlazeBolt.SpriteGetSize(entity)

Точка привязки

lua
BlazeBolt.SpriteSetOrigin(entity, originX, originY)

Значения от 0 до 1. (0.5, 0.5) — центр, (0, 0) — левый нижний угол.

Вращение

lua
BlazeBolt.SpriteSetRotation(entity, degrees)

Цвет

lua
BlazeBolt.SpriteSetColor(entity, r, g, b, a)

Значения от 0 до 1. По умолчанию (1, 1, 1, 1) — белый.

Текстура

lua
BlazeBolt.SpriteSetTexture(entity, texturePath)

Видимость

lua
BlazeBolt.SpriteSetVisible(entity, visible)

Пример

lua
function Start()
    local sprite = BlazeBolt.CreateSprite("icon.png", 0, 0)
    BlazeBolt.SpriteSetSize(sprite, 0.3, 0.3)
    BlazeBolt.SpriteSetOrigin(sprite, 0.5, 0.5)
    BlazeBolt.SpriteSetColor(sprite, 1, 0.5, 0, 1)
    BlazeBolt.SpriteSetRotation(sprite, 45)
end

◆Анимации

Анимированные спрайты из GIF-файлов или спрайт-листов

Создание из GIF

lua
entity = BlazeBolt.CreateAnimation(gifPath, true, x, y)

Создание из спрайт-листа

lua
entity = BlazeBolt.CreateAnimationFromSheet(texturePath, frameWidth, frameHeight, totalFrames, framesPerRow, frameDelayMs, x, y)
ПараметрТипОписание
texturePathstringПуть к спрайт-листу
frameWidthintШирина одного кадра (px)
frameHeightintВысота одного кадра (px)
totalFramesintОбщее количество кадров
framesPerRowintКоличество кадров в строке
frameDelayMsintЗадержка кадра (мс), по умолчанию 100
x, ynumberПозиция в NDC

Воспроизведение

lua
BlazeBolt.AnimationPlay(entity)
BlazeBolt.AnimationPause(entity)
BlazeBolt.AnimationStop(entity)        -- остановить, сброс на первый кадр
BlazeBolt.AnimationRestart(entity)     -- перезапустить с начала

Настройки

lua
BlazeBolt.AnimationSetLooping(entity, looping)  -- зацикливание (boolean)
BlazeBolt.AnimationSetSpeed(entity, speed)       -- множитель скорости (1.0 = норма)
BlazeBolt.AnimationSetFrame(entity, frameIndex)  -- перейти к конкретному кадру

Информация

lua
frameCount = BlazeBolt.AnimationGetFrameCount(entity)
isPlaying  = BlazeBolt.AnimationIsPlaying(entity)

Пример

lua
function Start()
    local anim = BlazeBolt.CreateAnimationFromSheet("player_sheet.png", 32, 32, 8, 4, 100, 0, 0)
    BlazeBolt.AnimationSetSize(anim, 0.4, 0.4)
    BlazeBolt.AnimationSetLooping(anim, true)
    BlazeBolt.AnimationSetSpeed(anim, 2.0)
end

Рендеринг текста через FreeType с поддержкой TTF-шрифтов

Создание

entity = BlazeBolt.CreateText(fontPath, text, x, y, fontSize)
ПараметрТипОписание

textstringТекст для отображения
x, ynumberПозиция в NDC
fontSizeintРазмер шрифта, по умолчанию 48

Содержимое

lua

Цвет, масштаб, позиция, видимость

lua
BlazeBolt.TextSetColor(entity, r, g, b, a)
BlazeBolt.TextSetScale(entity, scale)
BlazeBolt.TextSetPosition(entity, x, y)
BlazeBolt.TextSetVisible(entity, visible)

Пример

lua
function Start()
    fpsText = BlazeBolt.CreateText("fonts/arial.ttf", "FPS: 60", -0.8, 0.8, 24)
    BlazeBolt.TextSetColor(fpsText, 1, 1, 1, 1)
end

function Update(dt)
    local fps = math.floor(1 / dt)
    BlazeBolt.TextSetString(fpsText, "FPS: " .. fps)
end

◆Меши

Кастомная 2D геометрия с вершинами и индексами

Создание

lua
entity = BlazeBolt.CreateMesh()

Установка данных

lua
BlazeBolt.MeshSetData(entity, vertices, indices)

Каждая вершина — таблица: { x = , y = , u = , v = }

Отрисовка

lua
BlazeBolt.MeshDraw(entity)

Пример

lua
function Start()
    local mesh = BlazeBolt.CreateMesh()
    local vertices = {
        { x = -0.5, y =  0.5, u = 0, v = 1 },
        { x =  0.5, y =  0.5, u = 1, v = 1 },
        { x =  0.5, y = -0.5, u = 1, v = 0 },
        { x = -0.5, y = -0.5, u = 0, v = 0 }
    }
    local indices = { 0, 1, 2, 2, 3, 0 }
    BlazeBolt.MeshSetData(mesh, vertices, indices)
end

function Draw()
    BlazeBolt.MeshDraw(mesh)
end

◆Физика

Симуляция физики на базе Box2D — тела, коллизии, силы

Инициализация

lua
BlazeBolt.PhysicsInit(gravityX, gravityY)

Гравитация

lua
BlazeBolt.PhysicsSetGravity(x, y)
x, y = BlazeBolt.PhysicsGetGravity()

Создание тела

lua
entity = BlazeBolt.PhysicsCreateBody(bodyType, x, y, mass, friction, restitution)
ПараметрТипОписание
bodyTypestring"dynamic", "static", "kinematic"
x, ynumberПозиция в NDC
massnumberМасса тела
frictionnumberТрение (0–1)
restitutionnumberУпругость (0–1)

Формы (коллизии)

lua
BlazeBolt.PhysicsAddCircle(bodyEntity, radius, offsetX, offsetY)
BlazeBolt.PhysicsAddRectangle(bodyEntity, halfWidth, halfHeight)

Скорость

lua
BlazeBolt.PhysicsSetLinearVelocity(body, vx, vy)
vx, vy = BlazeBolt.PhysicsGetLinearVelocity(body)

BlazeBolt.PhysicsSetAngularVelocity(body, av)
av = BlazeBolt.PhysicsGetAngularVelocity(body)

Силы и импульсы

lua
BlazeBolt.PhysicsApplyForce(body, fx, fy)
BlazeBolt.PhysicsApplyForceAtPoint(body, fx, fy, px, py)
BlazeBolt.PhysicsApplyImpulse(body, ix, iy)
BlazeBolt.PhysicsApplyImpulseAtPoint(body, ix, iy, px, py)
BlazeBolt.PhysicsApplyTorque(body, torque)

Позиция и вращение

lua
BlazeBolt.PhysicsSetPosition(body, x, y)
x, y = BlazeBolt.PhysicsGetPosition(body)

BlazeBolt.PhysicsSetAngle(body, angle)
angle = BlazeBolt.PhysicsGetAngle(body)

Свойства тела

lua
BlazeBolt.PhysicsSetGravityScale(body, scale)    -- множитель гравитации
scale = BlazeBolt.PhysicsGetGravityScale(body)

BlazeBolt.PhysicsSetActive(body, active)         -- вкл/выкл тело
active = BlazeBolt.PhysicsIsActive(body)

BlazeBolt.PhysicsSetFixedRotation(body, fixed)   -- без вращения
fixed = BlazeBolt.PhysicsIsFixedRotation(body)

BlazeBolt.PhysicsSetBullet(body, bullet)         -- режим пули
bullet = BlazeBolt.PhysicsIsBullet(body)

mass = BlazeBolt.PhysicsGetMass(body)

Синхронизация и шаг

lua
BlazeBolt.PhysicsSyncSprite(bodyEntity, spriteEntity)  -- копирует pos + angle
BlazeBolt.PhysicsStep()                                    -- один шаг симуляции
Важно: Вызывайте PhysicsStep() и PhysicsSyncSprite() каждый кадр в Update(dt).

Пример

lua
function Start()
    BlazeBolt.PhysicsInit(0, -9.8)
    
    player = BlazeBolt.CreateSprite("icon.png", 0, 0)
    BlazeBolt.SpriteSetSize(player, 0.1, 0.1)
    
    body = BlazeBolt.PhysicsCreateBody("dynamic", 0, 0, 1, 0.3, 0.5)
    BlazeBolt.PhysicsAddCircle(body, 0.05, 0, 0)
end

function Update(dt)
    BlazeBolt.PhysicsStep()
    BlazeBolt.PhysicsSyncSprite(body, player)
    
    if BlazeBolt.IsKeyJustPressed(Keys.SPACE) then
        BlazeBolt.PhysicsApplyImpulse(body, 0, 5)
    end
end

◆Пример: мини-игра

Полный пример с физикой, управлением, сбором монет и звуком

lua
local playerSprite, playerBody
local score = 0
local scoreText
local coins = {}

function Start()
    BlazeBolt.SetMainWindowTitle("Mini Game")
    
    -- Игрок
    playerSprite = BlazeBolt.CreateSprite("player.png", 0, 0)
    BlazeBolt.SpriteSetSize(playerSprite, 0.08, 0.08)
    BlazeBolt.SpriteSetOrigin(playerSprite, 0.5, 0.5)
    
    -- Физика игрока
    playerBody = BlazeBolt.PhysicsCreateBody("dynamic", 0, 0, 1, 0.2, 0.6)
    BlazeBolt.PhysicsAddRectangle(playerBody, 0.04, 0.04)
    
    -- Земля
    local ground = BlazeBolt.CreateSprite("ground.png", 0, -0.85)
    BlazeBolt.SpriteSetSize(ground, 3, 0.05)
    local gb = BlazeBolt.PhysicsCreateBody("static", 0, -0.85, 0, 0.5, 0.3)
    BlazeBolt.PhysicsAddRectangle(gb, 1.5, 0.025)
    
    -- HUD
    scoreText = BlazeBolt.CreateText("fonts/arial.ttf", "Score: 0", -0.8, 0.85, 20)
    BlazeBolt.TextSetColor(scoreText, 1, 1, 1, 1)
    
    -- Звук
    BlazeBolt.LoadSound("music/bgm.mp3", "bgm", true)
    BlazeBolt.LoadSound("sfx/jump.wav", "jump", false)
    BlazeBolt.LoadSound("sfx/coin.wav", "coin", false)
    BlazeBolt.PlaySound("bgm")
    
    -- Физика
    BlazeBolt.PhysicsInit(0, -9.8)
    spawnCoins()
end

function spawnCoins()
    coins = {}
    for i = 1, 5 do
        local c = BlazeBolt.CreateSprite("coin.png",
            BlazeBolt.Random(-0.6, 0.6),
            BlazeBolt.Random(-0.5, 0.6))
        BlazeBolt.SpriteSetSize(c, 0.06, 0.06)
        table.insert(coins, c)
    end
end

function Update(dt)
    if BlazeBolt.IsKeyPressed(Keys.A) then
        BlazeBolt.PhysicsApplyForce(playerBody, -15, 0)
    end
    if BlazeBolt.IsKeyPressed(Keys.D) then
        BlazeBolt.PhysicsApplyForce(playerBody, 15, 0)
    end
    
    local _, vy = BlazeBolt.PhysicsGetLinearVelocity(playerBody)
    if BlazeBolt.IsKeyJustPressed(Keys.SPACE) and vy < 0.1 then
        BlazeBolt.PhysicsApplyImpulse(playerBody, 0, 8)
        BlazeBolt.PlaySound("jump")
    end
    
    BlazeBolt.PhysicsStep()
    BlazeBolt.PhysicsSyncSprite(playerBody, playerSprite)
    
    -- Сбор монет
    local px, py = BlazeBolt.PhysicsGetPosition(playerBody)
    for i = #coins, 1, -1 do
        local cx, cy = BlazeBolt.SpriteGetPosition(coins[i])
        if math.sqrt((px-cx)^2 + (py-cy)^2) < 0.08 then
            BlazeBolt.Destroy(coins[i])
            table.remove(coins, i)
            score = score + 10
            BlazeBolt.TextSetString(scoreText, "Score: " .. score)
            BlazeBolt.PlaySound("coin")
        end
    end
    
    if #coins == 0 then spawnCoins() end
    
    if BlazeBolt.IsKeyJustPressed(Keys.ESCAPE) then BlazeBolt.Quit() end
end

◆Лучшие практики

Рекомендации для эффективной разработки на BlazeBolt

Организация кода

  • Разделяйте логику на отдельные скрипты через scripts.list
  • Используйте BlazeBolt.EnableScript(name, false) для отладки
  • Храните пути к ресурсам в константах
  • Создавайте объекты в Start(), а не в Update()

Производительность

  • Не создавайте объекты в Update() — используйте пулы объектов
  • Вызывайте Destroy() для удаления ненужных объектов
  • Текстуры кешируются — SpriteSetTexture не загружает повторно
  • Используйте PhysicsStep() один раз за кадр

Масштабирование

  • Проектируйте в NDC — игра работает на любом разрешении
  • Для pixel-art используйте Nearest фильтр (по умолчанию)
  • Тестируйте на разных размерах через SetMainWindowSize

Отладка

  • BlazeBolt.Print() — вывод в консоль
  • 1 / BlazeBolt.GetDeltaTime() — текущий FPS
  • ReloadAllScripts() — перезагрузка без перезапуска
  • Проверяйте entity ~= 0 после создания

Обработка ошибок

lua
-- Проверка создания объекта
local sprite = BlazeBolt.CreateSprite("icon.png", 0, 0)
if sprite == 0 then
    BlazeBolt.Print("Ошибка: не удалось создать спрайт")
end

-- Проверка загрузки звука
local sid = BlazeBolt.LoadSound("sfx.wav", "name", false)
if sid >= 0 then
    BlazeBolt.PlaySound("name")
end
